# Venta App

1. Install dependencies

```shell
npm install
```

1. Seed database

```shell
npx prisma db push
```

```shell
npx prisma generate
```

```shell
npx prisma db seed
```

2. Build the app

```shell
npm run build
```

3. Start the app

```shell
npm run start
```

4. Open the app in the browser with the url: http://localhost:3002
